Number Guesser!

This game involves two players, you and the computer. For each round, there exist a secret target number between and including 0 to 9, both players are to guess what the number is before revealing it.

To make the game a little easier for both players,the player with the closest number to the secret target wins the round. Enjoy!!!

Classic

Contains 10 rounds
Tournament

Contains 5 stages
Endless

Unlimited rounds
